# covid-19 metrics
### all apps running in Docker:

- Grafana
- Influxdb
- custom Python collector

metrics are collected from: https://github.com/datasets/covid-19 aggregated daily.

run containers:  
```sh
sudo sh ./docker-compose-up.sh
```
